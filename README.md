# Projeto Apresentação Pessoal

Este projeto foi desenvolvido como exercício prático de Front-End utilizando **HTML5** e **CSS3**, com foco na criação de uma página de apresentação pessoal moderna e responsiva baseada em um protótipo no Figma.

A interface apresenta informações profissionais, descrição pessoal, links para redes sociais e uma ilustração personalizada, aplicando conceitos fundamentais de estruturação semântica, estilização, alinhamento com Flexbox e organização visual.

## Tecnologias utilizadas

* HTML5
* CSS3
* Google Fonts
* Flexbox
* Figma

## Funcionalidades

* Estrutura semântica em HTML
* Layout centralizado e responsivo
* Estilização moderna com CSS
* Integração com Google Fonts
* Espaçamento e alinhamento baseados em protótipo Figma
* Ícones de redes sociais
* Organização visual utilizando Flexbox

## Objetivo do projeto

O principal objetivo deste projeto foi praticar:

* Estruturação de páginas web
* Posicionamento de elementos
* Controle de espaçamentos
* Tipografia
* Responsividade
* Conversão de layout do Figma para código
